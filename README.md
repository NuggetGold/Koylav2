Описание
-------
Это простой Discord-бот, написанный на Java.

Требования
---------
- Java 17
- Maven
- Токен бота Discord (создаётся в Discord Developer Portal)
- Разрешённые intents (включая GUILD_MEMBERS)

Быстрый старт
------------
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/<ваш-юзер>/<repo>.git
   cd <repo>
   ```

2. Настройте конфиг.

3. Соберите проект:
   - Maven:
     ```
     mvn clean package
     ```

4. Запустите:
   ```
   запускай мейн класс
   ```


делайте с ним что хотите
